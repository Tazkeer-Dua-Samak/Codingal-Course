Data And Its Types

Data is any sequence of one or more symbols. It requires interpretation to become information.

Types:

1.  Text

2.  Numbers

3.  Images

4.  Audio

5.  Video

Text:

            Text consists of words, sentences and paragraphs. Text processing refers to the ability to edit manipulate text.

Numbers:

                       Numeric data consists of digits from 0 to 9, +(positive) or -(negative) sign and a decimal point. It can be integer type or real type data.
